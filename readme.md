## 🔗  How to use

### ⚙️   Launch the development server & meeting on http://localhost:3000/

### Project setup
```javascript
npm install
```

### Compiles for development
```javascript
npm run dev 
or 
npx vite
```

### Compiles and minifies for production
```javascript
npm run build
or 
npx vite build
```
